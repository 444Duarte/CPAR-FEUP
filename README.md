##Compiling
##Prerequisites
* [GCC, the GNU Compiler Collection](https://gcc.gnu.org/)
* [PAPI](http://icl.utk.edu/papi/)

###Compile
```
    gcc matrixprod.cpp -lstdc++ -lpapi -o matrixprod
```
